This page allows you to create new annotators and administrators.

Annotators vs. Administrators: While both annotators and administrators can annotate the presidents data, only annotator contributions will be included for analysis in the downloadable data frames. In contrast, administrators are able to track annotators' progress, download data frames, and create and delete users. (Thus, if you're viewing this page, this means you're an administrator!)

To create a new user: Enter the new user's username and email in the fields below and select whether the user is an annotator or administrator. Upon clicking Create User, a link will be sent to the new user, allowing them to set up their password. And, voila!
